# üìö UniGe Project: LLM Data Augmentation for Text Classification

This repository contains the source code and experimental results for the project **"Using LLMs for Data Augmentation in Text Classification Tasks"**.

## üìù Project Overview

The goal of this project is to answer the research question: *Can we leverage Large Language Models (LLMs) to improve the performance of lightweight classifiers in data-scarce environments?*

We compare **Support Vector Machines (SVM)** and **Random Forests (RF)** on a Spam Detection task (SMS Spam Collection) under two conditions:
1.  **Baseline:** Trained on a tiny dataset of only 40 real samples.
2.  **Augmented:** Trained on the same 40 samples + synthetic data generated by **Google Gemini**.
